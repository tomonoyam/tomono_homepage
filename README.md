# Concept

AUTOSARは難しい。

- 用語が多い
- レイヤが深い
- 構成が巨大
- 全体像が見えづらい

しかし、
その構造には確かな「思想」があります。

このサイトでは、
AUTOSARを魔法のような比喩や世界観を交えながら、
本質的に理解することを目的としています。
